# Internship Placement Analytics Dashboard (Synthetic Data)

This project uses a synthetic dataset to explore internship placement trends, sourcing patterns, programme performance, and company-level insights. The real dataset cannot be shared due to confidentiality.

## Visualisations
- Placement count by year
- Intern source type over time
- Programme conversion trends
- Company-level breakdown

## Tools Used
Power BI, Excel/CSV, data modelling, visual storytelling

## Purpose
To demonstrate how data can support decision-making in academic operations and to build my analytics portfolio.
